🎉 超簡單使用說明 - 連小朋友都會用！

═══════════════════════════════════════════════════════════════

📥 第一步：下載文件

1. 打開瀏覽器（Chrome、Edge、Firefox 都可以）
2. 在網址列輸入：https://github.com/lf3lk3123/aaa
3. 找到綠色的 "Code" 按鈕（很大很明顯）
4. 點擊 "Download ZIP"
5. 文件會下載到您的"下載"文件夾
6. 找到下載的 ZIP 文件，右鍵點擊選擇"解壓縮"
7. 解壓縮到桌面（這樣比較好找）

═══════════════════════════════════════════════════════════════

🐍 第二步：安裝 Python（只需要做一次）

1. 打開新的瀏覽器分頁
2. 輸入：https://www.python.org/downloads/
3. 看到黃色的 "Download Python" 按鈕，點它
4. 下載完成後，雙擊安裝文件
5. ⚠️ 重要：安裝時一定要勾選 "Add Python to PATH"
6. 然後一直點"下一步"直到安裝完成

═══════════════════════════════════════════════════════════════

🚀 第三步：運行程式

1. 到桌面找到剛才解壓縮的文件夾（名字叫 aaa-main 或類似）
2. 雙擊打開文件夾
3. 找到 "快速啟動.bat" 文件（有個齒輪圖標）
4. 雙擊這個文件
5. 會彈出一個黑色視窗，顯示選項
6. 輸入數字 1，然後按 Enter 鍵
7. 等待幾秒鐘，程式就會自動打開！

═══════════════════════════════════════════════════════════════

🎯 如果遇到問題：

問題1：找不到 "快速啟動.bat"
解決：確保您解壓縮了整個文件夾，不是只解壓縮部分文件

問題2：雙擊 .bat 文件沒反應
解決：右鍵點擊文件，選擇"以系統管理員身分執行"

問題3：出現 "Python 未找到" 錯誤
解決：重新安裝 Python，記得勾選 "Add Python to PATH"

問題4：程式打開後是英文
解決：程式支援中文，如果顯示英文可能是系統設定問題
